The compareDocumentPosition() Method

This is a paragraph

This is another paragraph

The position of the two paragraphs is:

Possible values:
1: The two nodes do not belong to the same document.
2: p1 is positioned after p2.
4: p1 is positioned before p2.
8: p1 is positioned inside p2.
16: p2 is positioned inside p1.
32: The two nodes are attributes on the same element.

The return value can also be a combination of values. Return value 20 means that p2 is inside p1 (16) AND p1 is positioned before p2 (4).
